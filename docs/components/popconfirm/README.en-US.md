### API

#### Props

| 参数        | 说明         | 类型   | 默认值 |
| ----------- | ------------ | ------ | ------ |
| title       | 显示内容     | string |        |
| confirmText | 确认按钮文本 | string |        |
| cancelText  | 取消按钮文本 | string |        |

#### Events

| 参数      | 说明         | 类型     | 默认值 |
| --------- | ------------ | -------- | ------ |
| onCancel  | 点击关闭回调 | function |        |
| onConfirm | 点击确认回调 | function |        |
