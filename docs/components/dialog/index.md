### API

| 参数        | 说明             | 类型     | 默认值  |
| ----------- | ---------------- | -------- | ------- |
| title       | 标题             | string   | default |
| show-button | 是否显示底部按钮 | boolean  | true    |
| onClose     | 关闭 Dialog      | function |         |
